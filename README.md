# IQSkins

A simple pixel-style web template for a gaming skin exchange. This skeleton demonstrates how a marketplace for Counter Strike and Rust skins could look. It does not provide real trading functionality but gives a base to build upon.

## Getting Started

Open `index.html` in a browser or serve the folder locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` to view the site.
